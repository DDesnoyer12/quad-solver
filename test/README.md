# Make Commands to Run Each Test Case
  Run these make commands while in directory ../src after you run make to build the program.
  ## Solution Set with 1 answer
    make test-1set

  ## Solution Set with 2 answers
    make test-2set
  
  ## Double Bounds
    make test-bounds
  
  ## If a,b, or c is Inf
    make test-inf
    
  ## If a,b, or c is NaN
    make test-nan
    
  ## If a,b, or c is not Normal
    make test-norm
    
  ## Generic Test
    make test
